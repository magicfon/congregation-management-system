# 會眾管理系統 - 項目原則

## 質量標準
- TypeScript 嚴格模式
- ESLint + Prettier 代碼規範
- 單元測試覆蓋率 > 80%

## 可維護性
- 清晰的模組化架構
- API 路由統一規範
- 組件複用性

## 用戶體驗
- Mission Control 深色主題
- 響應式設計
- 無障礙訪問

## 數據安全

- bcrypt 密碼加密
- NextAuth.js 認證

## 技術棧
- Next.js 14 + TypeScript
- Prisma + SQLite
- Tailwind CSS
- NextAuth.js
